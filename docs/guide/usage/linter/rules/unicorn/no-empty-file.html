unicorn/no-empty-file Correctness ​

✅ This rule is turned on by default.

What it does ​

Disallows any files only containing the following:

  • Whitespace
  • Comments
  • Directives
  • Empty statements
  • Empty blocks
  • Hashbang

Why is this bad? ​

Meaningless files clutter a codebase.
